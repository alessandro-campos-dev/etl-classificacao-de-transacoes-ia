# Fluxograma do ETL

1. **Extract**
   - Monta Google Drive
   - Lê CSV original

2. **Transform**
   - Limpa textos
   - Padroniza datas
   - Converte valores

3. **Classify**
   - Regras determinísticas
   - IA Generativa
   - Fallback heurístico

4. **Load**
   - Exporta CSV final
   - Disponibiliza download
